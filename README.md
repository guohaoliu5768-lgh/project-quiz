# 项目投资监管学习测验

这是一个基于项目文件夹现有学习材料整理的静态网页测验程序。

## 使用方式

- 打开 `index.html` 即可开始测验。
- 支持普通、中等、困难三种难度。
- 每次测验 30 题。
- 内置题库共 210 题，普通、中等、困难各 70 题。
- 题库按现有材料覆盖审批权限、立项资料、招投标、合同、验收、稽核、后评估、信息化和授权额度等知识点。
- 每次点击“开始测验”时从当前难度题库中随机抽取一次 30 题。
- 每题的正确答案会随机分布在 A/B/C/D 选项中，避免固定出现在同一选项。
- 困难模式的错误选项保持在同一知识点范围内，避免明显无关的干扰项。
- 每题确认后显示正确答案和解析。
- 测验完成后生成答题情况分析报告和错题回顾。

## 部署方式

本项目适合通过 GitHub Pages 静态托管。仓库建议命名为 `project-quiz`，发布后访问地址通常为：

```text
https://<GitHub用户名>.github.io/project-quiz/
```

## 发布内容

线上发布只需要以下文件：

- `index.html`
- `README.md`
- `.nojekyll`

原始 PDF、PPTX 学习材料不需要上传到公开仓库。
